1. Открыть Visual Studio Installer
2. Убедиться что скачаны:
 - Разработка приложений для универсальной платформы (значок виндовса)
 - ASP.NET и разработка веб приложений (понадобится для 10 домашнего задания).
3. Клонировать репозиторий по ссылке командой git clone "ссылка".
4. Ветки для домашки именовать как feature/Homework№
5. Ветка создается из master с помощью команды git checkout -b "Name". То есть перед написанием команды вы
должны находиться  на ветке master.
6. В ветке выполняется домашнее задание.
6.1. Внутри репозитория создается проект(консоль, библиоткеи классов и т.д.). Также создается README.md файл куда копируется условие домашнего задания.
6.3. Внутри solution выполняется домашняя работа.
7. Делается коммит командой git commit -m "Сообщение".
8. Затем push в github.
9. Создается пул реквест из ветки задачи в ветку мастер.
10. Если апрув, то сливается в ветку master.
11. Локально переключаетесь на ветку master и делаете git pull в консоли.
12. Затем из master опять создается ветка для следующего домашнего задания и т.д.

Все гитовые операции можно делать и в visual studio, но для первого раза рекомендую всё делать через консоль
чтобы понять как это устроено, дальше уже можно всё делать в Visual Studio.
